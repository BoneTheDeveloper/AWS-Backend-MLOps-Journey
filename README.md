# AWS Backend MLOps Journey

A structured learning path from AWS Cloud Foundation to Production-Ready MLOps.

## Project Structure

```
AWS-Backend-MLOps-Journey/
├── Phase-1-Cloud-Foundation/   # AWS SAA notes and simple backend projects
├── Phase-2-MLOps-Bridge/       # Containerized models and Docker files
├── Phase-3-Production/         # Terraform scripts and CI/CD pipelines
└── README.md                   # This file
```

## Phases Overview

### Phase 1: Cloud Foundation
- AWS Solutions Architect Associate study notes
- Simple backend API projects on AWS
- Core services: EC2, S3, RDS, Lambda, API Gateway

### Phase 2: MLOps Bridge
- Docker containerization of ML models
- Model serving with Docker
- Introduction to container orchestration concepts

### Phase 3: Production
- Infrastructure as Code with Terraform
- CI/CD pipelines for ML models
- Monitoring, logging, and observability

## Getting Started

Each phase is self-contained. Navigate to the phase directory you're interested in and follow the instructions within.
